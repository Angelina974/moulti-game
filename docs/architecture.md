# Architecture

## Frontend
- `frontend/index.html`: entry point
- `frontend/pages/`: page templates and game pages
- `frontend/assets/`: static assets (css, images, audio, videos)
- `frontend/js/`: shared frontend scripts

## Backend
- `backend/server.js`: API/server entry point
- `backend/src/`: backend source modules

## Database
- `database/schema.sql`: SQL schema
- `database/seed.sql`: seed data
